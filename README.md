# Arulraj Veerappan - Portfolio Website

A professional, modern, entirely responsive personal portfolio crafted for a Front-End Developer.

## Overview
Built with clean architecture, emphasizing CSS Flexbox and Grid without any external CSS/JS libraries or frameworks, except Native Service Worker for PWA capabilities.

## Technical Details
- **UI:** HTML5, CSS3 
- **PWA functionality:** JavaScript (Service Workers)
- **Features:** fully responsive, dark theme, smooth transitions, offline fallback mechanism.

## Deployment
Simply expose the `portfolio/` directory using any static web server (e.g., Live Server, Python `http.server`, standalone deployment to Vercel/Netlify). No build tooling requires execution.
